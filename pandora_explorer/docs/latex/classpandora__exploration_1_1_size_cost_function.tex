\subsection{\-Detailed \-Description}
\-A class implementing a frontier cost function using the provided interface. 

\-This cost function adds costs to the frontiers based on the frontier size. \-Frontiers bigger in size are added higher costs. \-Cost are normalized to a scale between 0 and 1.
